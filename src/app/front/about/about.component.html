About

This is the web app for the project titled 'RFID Class Attendance System'. It is built with Angular2 using Typescript. The app is served by a nodejs app which also provides API routes for the hardware implementation.

This App provides an interface for students and lecturers to login, signup. It also enables students to keep track of their, class attendance records for specific courses. Similarly, lecturers can view records for courses they co-ordinate and download relevant data.
